# algorithmic-puzzles

## Install

1. Install [Stack](https://docs.haskellstack.org/en/stable/README/)
2. Setup dependencies
```bash
stack setup
```

## Build

```bash
stack build
```

## Run an executable

001. A Wolf, a Goat, and a Cabbage
```bash
stack exec 001-wolf-goat-cabbage
```

002. Glove Selection
```bash
stack exec 002-glove-selection
```
